# EnglishPractice App

## RUN : 
    1. For ServerSideApp:
      -open terminal
      -cd EnglishPracticeApp
      -run the command : (docker-compose up)
    2. For ClientSideApp:
      -open new terminal
      - run the command (npm install @angular/cli)
      -cd EnglishPracticeApp/ClientSideApp
      -run the command (ng serve)
## USE : 
    - open this link in browser (http://localhost:4200/)
    
## TEST :
